# OOP-Snake-Game-with_CSharp
A desktop-based Snake Game developed in C# using Object-Oriented Programming (OOP) and Windows Forms (.NET Framework). Features a responsive GUI, modular architecture, JSON-based score storage, and a fully functional executable (.exe), demonstrating game development, event-driven programming, and file handling.

**Features**
Classic Snake gameplay with keyboard controls
Responsive Windows Forms graphical user interface
Object-Oriented Programming (OOP) implementation
JSON-based data storage for scores and game information
Event-driven game logic and collision detection
Standalone executable (.exe) application
**Technologies Used**
C#
.NET Framework
Windows Forms (WinForms)
Object-Oriented Programming (OOP)
JSON Serialization and File Handling
Project Structure
Frontend: Windows Forms (.NET Framework)
Backend: C# classes implementing game logic and OOP concepts
Data Storage: JSON files
Executable: Ready-to-run .exe application
**How to Run**
Clone or download the repository.
Open the solution in Visual Studio.
Build and run the project.
Alternatively, run the included executable (.exe) file.
